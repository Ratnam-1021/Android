# ScreenTracker - Jetpack Compose

An Android app that monitors and visualizes daily device usage using **UsageStatsManager** and **Vico Charts**.

## ?? Features
- **Usage Permissions**: Detects and requests system-level usage access.
- **Data Visualization**: Top 5 apps displayed in a dynamic Bar Graph.
- **Detailed Log**: Breakdown of every app used today in minutes.

## ?? Tech Stack
- **UI**: Jetpack Compose
- **Charts**: Vico Library
- **API**: Android UsageStatsManager
