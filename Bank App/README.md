# BankApp - Jetpack Compose

A functional Android Banking application built with **Kotlin** and **Jetpack Compose**. This project demonstrates modern Android development practices, including state management and declarative UI.

## ? Key Features
* **Secure Login**: Entry system using pre-configured user accounts.
* **Dynamic Dashboard**: Displays account holder name and real-time balance updates.
* **Financial Transactions**: Support for Deposits and Withdrawals with instant balance reconciliation.
* **Account Transfers**: Ability to transfer funds between different users within the app state.
* **Transaction History**: A detailed log of all activities, categorized by type (Deposit, Transfer, etc.) with date stamps.

## ??? Technical Implementation
* **Jetpack Compose**: For a fully declarative UI.
* **Navigation Compose**: Handles the flow between Login, Dashboard, and Transaction screens.
* **State Management**: Uses mutableStateOf and mutableStateListOf to ensure the UI stays in sync with data.
* **Kotlin**: Leveraged for clean, concise business logic.
